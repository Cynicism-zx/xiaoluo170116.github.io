Linux操作系统1.连接云服务器 2.日常操作 ： 1）cd指令 cd 目录 —— 进入指定目录 cd .. 返回上层目录 cd ~ 回到根目录 cd / 回到系统根目录 2）ls指令 ls —— 查看当前目录 ls -l/-lh —— 查看详情 ls -a ——显示所有文件 ls -R —— 递归显示所有内容 ls -S/-t —— 按大小/时间排序 注意：多个功能不冲突的参数可以同时使用，中间用空格隔开，例如：ls -t -h 3）pwd指令 —— 显示当前完整目录（路径） 4）文件操作指令 touch 文件名 —— 新建文件 cat 文件名 —— 查看文件内容（只能可读） Vim/vi 文件名 —— 打开文件（可读可操作，文件不存在会创建再打开） rm 文件名 —— 删除文件（询问是否删除吧） rm -r —— 删除文件夹（提醒是否删除） rm -f 目录 —— 强制性删除文件
